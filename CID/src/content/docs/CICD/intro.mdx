CI/CD står for Continuous Integration/Continuous Deployment, og er en praksis som brukes for å automatisere bygging, testing og utrulling av programvare.

CI/CD er en viktig del av DevOps, som er en tilnærming til programvareutvikling som fokuserer på samarbeid mellom utviklere og driftspersonell.

Ved å bruke CI/CD kan teamet ditt raskt og effektivt levere programvare av høy kvalitet til brukerne.
Dette kan bidra til å redusere feil, øke produktiviteten og forbedre brukeropplevelsen.

I dette kurset vil du lære om CI/CD-prinsippene, verktøyene som brukes og hvordan CI/CD blir brukt i arbeidslivet.
Vi vil også se på beste praksis og vanlige utfordringer som kan oppstå underveis.
